# Rock-Paper-Scissors Game

A simple interactive Rock–Paper–Scissors game built with vanilla JavaScript. The user selects a move, the computer generates a random choice, and the game determines and displays the result based on standard rules.

## Features
- Player vs computer gameplay
- Randomized computer choice
- Real-time result display (win, lose, or draw)
- Simple and intuitive user interface

## Tech Stack
- HTML
- CSS
- JavaScript

## Purpose
Built as a practice project to reinforce core JavaScript concepts such as conditionals, random number generation, event handling, and DOM manipulation.
